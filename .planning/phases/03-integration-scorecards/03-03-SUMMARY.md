## Decisions Made

- `tag=tool-completion` on all inline tool forms to allow Buttondown segmentation from landing page subscribers (`tag=landing`)
- Compass scorecard changed from `chartType: "radar"` to `"bar"` for the config passed to `generateScorecardPreview` — both types render, bar more readable at 600x314 thumbnail size
- Flow Logger scorecard preview updates on every activity add/delete (no single-shot "results" event exists in this persistent tool)
- Preview is generated at results render time rather than lazily on scroll, so it is immediately visible without interaction
