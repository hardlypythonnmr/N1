**renderCompletionBadges(profile)**
For each of [values, strengths, identity, purpose]:

- Create a `<span class="tag">` element
- If `profile[dim].completed === true`: add class `tag-orange`, text = dimension name in Spanish (Valores, Fortalezas, Identidad, Propósito)
- Else: style with `opacity: 0.4`, text = dimension name + " —"
- Append to `#completion-badges`

**renderRadarChart(profile)**
Extract dimension averages:

- values: average of non-null Schwartz values (schwartz_profile object), scale 1-9 → normalize to 0-10
- strengths: count of signature_strengths (max cap at 10 for display) OR flow_activities count; use whichever is present
- identity: average of adaptability (concern, control, curiosity, confidence), scale 1-7 → normalize to 0-10
- purpose: average of mlq.presence + mlq.search, scale 1-7 → normalize to 0-10

If a dimension has no data: use value 0 and grey out that axis.

**renderInsights(profile)**
Reinforcement detection algorithm:

- If values.completed AND strengths.completed:
  - Check if top core_values overlap thematically with signature_strengths (simple string matching — if any value string shares a word with any strength string, count as alignment)
  - If alignment found: create a `.disclaimer` div (styled as insight, not warning) with text: "Refuerzo: tus valores y fortalezas apuntan en la misma dirección."
  - Else: create neutral note

Conflict detection:

- If triage.career_path === 'stay_grow' AND identity.adaptability.control < 3.5 (scale 1-7):
  - Show: "Atención: tu nivel de agencia (Control CAAS) es bajo — puede dificultar el crecimiento donde estás."
- If triage.career_path === 'go_independent' AND values.schwartz_profile.security > 7:
  - Show: "Tensión: valorás mucho la seguridad pero considerás la independencia — esto merece exploración."

Append findings to `#reinforcements-section` and `#conflicts-section` using safe DOM methods. If profile has no completed dimensions, show: "Completá al menos dos herramientas para ver insights transversales."
