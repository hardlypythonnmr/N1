**Tool 1: CV Audit**

  - stay_grow → internal resume/promotion case: emphasis on internal achievements, scope growth, executive presence
  - move → external resume: ATS optimization, transferable skills narrative, target role fit

- Cite: "Un estudio de Hiring Lab (2023) encontró que el 75% de los CVs son eliminados por ATS antes de llegar a un humano."
- Output: score por dimensión + 3 acciones concretas prioritizadas

**Tool 2: Interview Prep**

- Path-specific focus:
  - stay_grow → promotion conversation: internal interview format, political navigation, sponsorship
  - move → external interview: STAR stories from mastery experiences, company research integration
  - go_independent → pitch deck / consulting sales conversation
  - pause_reset → re-entry narrative, gap explanation, energy/motivation evidence
- If profile available: generate STAR stories from strengths.self_efficacy.mastery_experiences
- Practice: offer to role-play 2-3 most likely questions based on target role description
- Cite: "Bandura (1977): la autoeficacia se desarrolla principalmente a través de experiencias de dominio — practicar la entrevista no es vanidad, es construcción de confianza."

**Tool 3: Negotiate**

- Path-specific focus:
  - stay_grow → salary raise: internal leverage, timing, market data framing
  - move → offer negotiation: total comp breakdown, multiple offer leverage, walkaway point
  - go_independent → rate setting: market positioning, value-based pricing, scope creep prevention
  - pause_reset → re-entry terms: negotiating flexible arrangements, title matching, onboarding conditions
- LATAM context (from latam-market-context.md): salary opacity, reference ranges when available, employer-side dynamics
- If profile available: anchor negotiation rationale in values.core_values (autonomy, security, etc.)
- Never give a number without the user providing their market data first
- Cite: "Babcock & Laschever (2003): negociar el primer salario tiene un impacto acumulado de $500K-$1M+ en ingresos totales."

**Tool 4: STAR Builder**

- Collect raw achievement stories from the user (3-5 stories)
- Structure each story: Situation → Task → Action → Result with impact metrics
- If profile available: draw stories from strengths.self_efficacy.mastery_experiences; align with values
- Provide 3 finished STAR stories ready to paste into resume or use in interviews
- Quality check: each story must have a quantified result or clear impact

**Tool 5: Profile Builder (LinkedIn)**

  - stay_grow → internal visibility optimization

- Colombian/LATAM context: English vs Spanish profile strategy, international vs local job market

**Tool 6: Network Map**

- Granovetter weak ties framework (internal): map strong ties, weak ties, structural holes
- Identify 3-5 specific people to reconnect with based on career_path
- Path-specific outreach templates (each adapted to the relationship)
- Anti-networking framing: "No estamos haciendo networking — estamos teniendo conversaciones con personas que te conocen y cuyo criterio respetás."
- If profile available: match network targets to possible_selves or target industries from identity dimension
- Cite: "Granovetter (1973): el 83% de las oportunidades de trabajo provienen de vínculos débiles — conocidos, no amigos cercanos."

**Tool 7: Job Hunt**

- Channel strategy by career_path + values + LATAM context (from latam-market-context.md)
- Search funnel: target role identification → company list → channel mix → weekly action plan
- ATS vs direct application vs referral strategy
- Job posting quality check: reference ghost-check signals from assessment-scoring.md
- If profile available: filter roles by values alignment + identity.possible_selves

**Tool 8: Layoff Response Kit — 72-hour protocol**

- Hour 0-24: emotional processing acknowledgment → immediate financial safety check (no decisions yet) → practical logistics (benefits, COBRA/insurance equivalent, severance review)
- Hour 24-48: network activation (who to tell, how, in what order) → LinkedIn signal update → recruiter outreach sequence
- Hour 48-72: job search strategy → 10 target companies → first 3 outreach messages drafted
- Safety check: if user shows Tier 2 MIND-SAFE signals (burnout collapse, overwhelm), offer Tier 2 response before protocol
- Cite: "Las primeras 72 horas después de un layoff determinan en gran parte la velocidad de la transición — la investigación muestra que la activación temprana de red es el predictor más fuerte de reempleo rápido."

**Tool 9: Grow Where You Are**

- Internal opportunity scan: org chart awareness, budget cycles, project adjacencies
- Promotion case builder: track record documentation, strategic narrative, sponsorship map
- Manager conversation prep: how to have the promotion conversation, timing, what to ask for
- Environment optimization: negotiate remote/flexible arrangements, team transfer, scope expansion
- If profile available: align grow goals with values.bulls_eye (close the work alignment gap)
- Anti-framing: never assumes "stay" is the second choice — treat it as equally valid to leaving
- Cite: "Un estudio de McKinsey (2022) encontró que el desarrollo interno es el predictor más fuerte de retención a 3 años."

**File 2: latam-market-context.md**

Sections:

- Salary context: salary opacity in Colombia/LATAM; best available reference sources (Glassdoor Colombia limitations, LinkedIn Salary, sector-specific ranges in USD for tech roles: junior SWE, mid, senior, staff, engineering manager — approximate ranges for Bogota outsourcing firms); acknowledge data limitations explicitly
- ATS systems in LATAM: which ATSs are common (Workday, SAP SuccessFactors, Oracle Taleo), formatting that passes vs. fails ATS; LinkedIn Easy Apply dynamics in LATAM

- Structural barriers to name and validate (never dismiss): family/financial obligations, limited mobility, compensation compression vs. international market, informal economy transitions

**File 3: assessment-scoring.md**

Sections:

- CV Audit rubric (10 dimensions):
  1. Impact metrics: does each bullet have a quantified result? (0-10)
  2. ATS keyword density: role-specific keywords in first half of CV? (0-10)
  3. Narrative coherence: does the CV tell a logical career progression story? (0-10)
  4. Length/format: appropriate length (1 page for <5 years, 2 pages for >5), clean formatting (0-10)
  5. Headline/summary: compelling, specific, not generic? (0-10)
  6. Skills match: explicit match to target role requirements? (0-10)
  7. Value alignment: does the CV signal what the person values? (0-10, optional but powerful)
  8. Strength evidence: signature strengths visible in achievement descriptions? (0-10, optional)
  9. Cultural calibration: appropriate for target market (LATAM vs international)? (0-10)
  10. Contact/CTA: LinkedIn URL, location, clean contact info? (0-10)
  - Scoring: 80-100 = ready to send; 60-79 = one revision; <60 = significant work needed
  - Per-dimension improvement actions (1-2 sentences each)

- Ghost Check signal library (for Job Hunt tool use):
  - Red flags: posting age >90 days, no salary range, reposted multiple times, generic role title, vague company info, requests unusual info upfront, no specific team/manager, mass posting same day
  - Yellow flags: salary range very wide (3x+), location "flexible" without explanation, "competitive salary" with no range, unclear reporting structure
  - Green flags: specific salary range, named hiring manager, specific team context, detailed role requirements, recent posting (<30 days), company profile active

- Interview quality signals (for Interview Prep tool):
  - Strong signals from interviewer: specific role context, team introduction, clear timeline, asks behavioral questions
  - Weak signals: "we're still defining the role", can't explain day-to-day, no timeline, only asks generic questions
  - Red flags: asks about salary history (illegal in some contexts), vague team structure, no mention of offer process

Note: these files are AI reference documents — no user-facing text required. Technical/reference tone is appropriate.
