**File 1: values-framework.md**
Sections:

- Framework overview: ACT values clarification (Hayes et al. 1999) + Schwartz Basic Values (Schwartz 1992) + SDT self-direction (Deci & Ryan 1985)
- ACT exercises: values elicitation questions (exact Spanish voseo text from prompting-stack.md Section 1), defusion moves (internal), committed action prompts

- Bull's Eye scoring guide: 4 domains (work, relationships, personal_growth, leisure), scoring rubric 1-10, what gap scores mean
- Value conflicts: how to identify and name conflicts between competing Schwartz values; 3 example conflict patterns common in LATAM (security vs self-direction, benevolence vs achievement, tradition vs stimulation)
- Career implications by value profile: high self-direction → independence fit; high security → risk calibration; high achievement → promotion path; etc.
- Profile YAML mapping: which questions populate which fields (values.core_values, values.schwartz_profile._, values.bulls_eye._, values.value_conflicts, values.career_implications)

- Deep mode script: 15-25 turn Socratic sequence, homework suggestions (journaling prompt, values card sort description)

**File 2: strengths-framework.md**
Sections:

- Framework overview: VIA Character Strengths (Peterson & Seligman 2004) + Bandura Self-Efficacy (Bandura 1977) + Csikszentmihalyi Flow + Big Five (IPIP-NEO)
- VIA guide: 24 character strengths grouped by virtue (Wisdom, Courage, Humanity, Justice, Temperance, Transcendence), conversational elicitation for top 5 without administering full survey; reference viacharacter.org for free formal assessment
- Bandura 4 sources: mastery experiences (past successes), vicarious models (observed role models), social persuasion (feedback from credible others), physiological state — question sequence for each source
- Flow mapping: challenge-skill balance, conditions for flow, how to identify flow activities conversationally; reference web/flow-logger for longitudinal tracking
- Big Five: verbal approximation questions for each trait (O, C, E, A, N); scoring heuristic; how each trait maps to career contexts (e.g., high O → entrepreneurial roles, high C → execution roles)
- Integration: how VIA + Bandura + Flow intersect — signature strengths usually appear in flow activities and high mastery experiences
- Profile YAML mapping: strengths.signature_strengths, strengths.big_five._, strengths.self_efficacy._, strengths.flow_activities
- Quick mode and deep mode scripts

**File 3: identity-framework.md**
Sections:

- Framework overview: Ibarra Working Identity (Ibarra 2003) + Marcia Identity Status (Marcia 1966) + Savickas CAAS-12 (Savickas 2012)
- Ibarra: working identity as a set of possible selves tested through experiments, not introspection; how to identify current possible selves; how to design identity experiments ("¿Qué proyecto pequeño podrías hacer en los próximos 30 días que te acerque a [identity]?"); re-crafting narratives as identity work
- Marcia status heuristic: conversational assessment of 4 statuses (achieved/moratorium/foreclosed/diffused) — which questions reveal each status, how to name the status gently without clinical language; IMPORTANT: this is NOT a clinical diagnosis — say "tu exploración de identidad parece estar en una fase de [description]" not "tenés identity diffusion"
- CAAS-12 conversational: 4 subscales (Concern, Control, Curiosity, Confidence) — 3 questions per subscale for conversational approximation; explicit caveat that this is not the standardized instrument; reference web/compass for validated scores; lower control scores are normative in collectivist cultures (LATAM calibration)
- Possible selves: exercises to surface 3-5 possible professional identities; how to evaluate each against values profile; crafting small experiments for each
- Narrative theme: Savickas life theme extraction — 3 questions that surface the career story's underlying theme; how to reflect it back
- Limiting beliefs: how to surface and gently challenge identity-limiting beliefs using CBT moves (internal) without labeling them; common LATAM patterns (impostor syndrome, age limits, geographic constraint)
- Profile YAML mapping: identity.identity_status, identity.adaptability.\*, identity.possible_selves, identity.active_experiments, identity.narrative_theme, identity.limiting_beliefs
- Quick mode and deep mode scripts

**File 4: purpose-framework.md**
Sections:

- Framework overview: Frankl Logotherapy (Frankl 1946) + Steger MLQ (Steger 2006) + Ryff PWB (Ryff 1989) + Damon Purpose Development (Damon 2008)
- Frankl 3 meaning sources: creative (what you give to the world), experiential (what you receive — beauty, love, connection), attitudinal (your stance toward unavoidable suffering) — question sequence for each; how to identify the primary source for this person
- MLQ interpretation: presence subscale (1-7, current sense of meaning), search subscale (1-7, active seeking of meaning); 4-quadrant placement (fulfilled: high presence + low search; growing: high presence + high search; seeking: low presence + high search; disengaged: low presence + low search); LATAM calibration — high search is normative, not pathological; reference web/mlq for standardized scoring
- Ryff PWB purpose subscale: purpose in life as a dimension of psychological well-being; conversational items
- Damon: purpose as a stable intention to accomplish something meaningful beyond the self; how to distinguish purpose from goals; the "beyond the self" component as a marker
- Life theme extraction: Savickas life theme questions applied to purpose (what problem were you born to solve?); how to hold a life theme lightly as a working hypothesis, not a fixed identity
- Purpose practice: daily/weekly anchoring practices that maintain connection to purpose during routine work
- Career implications: how MLQ quadrant maps to career urgency and intervention type (disengaged → values work first; seeking → purpose exploration; fulfilled → optimization; growing → support the ongoing search)
- Profile YAML mapping: purpose.mlq._, purpose.meaning_sources._, purpose.life_theme, purpose.purpose_practice, purpose.career_implications
- Quick mode and deep mode scripts
