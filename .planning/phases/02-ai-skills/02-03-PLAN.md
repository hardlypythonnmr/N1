**Skill body:**

1. `<session_start>`:
   - MIND-SAFE check on opening message (brief version — Tier 1 and Tier 2 signals)
   - Scope disclaimer (verbatim from safety-protocol.md Section 1, Spanish version)
   - Profile check: "¿Tenés un perfil N1? Si lo tenés, pegalo y agrego un análisis de alineación con tus valores. Si no, igual puedo analizar la oferta."
   - Input request: "Pegá el texto completo de la oferta, o si no podés, describímela: empresa, rol, salario (si aparece), cuándo fue publicada, y cualquier detalle que te llame la atención."

2. `<scoring>` — ghost detection engine:

Signal categories and weights:

- **Posting age** (20 pts): < 30 days = 20, 30-60 days = 10, 60-90 days = 5, > 90 days = 0
- **Salary transparency** (20 pts): specific range = 20, "competitive" with no range = 5, no mention = 0
- **Role specificity** (15 pts): specific team + manager context + day-to-day = 15, vague = 5, generic = 0
- **Company presence** (15 pts): named company with active profile + recent activity = 15, vague company = 5, anonymous = 0
- **Language patterns** (15 pts): concrete requirements + specific outcomes = 15, buzzword-heavy + vague = 5, both = 8
- **Reposting signals** (15 pts): first posting = 15, reposted or duplicate detected = 3, uncertain = 8

Score tiers:

- 80-100: Alta probabilidad de ser real. Vale la pena aplicar con energía.
- 60-79: Señales mixtas. Aplicá pero con expectativas calibradas — es posible que sea real.
- 40-59: Señales preocupantes. Intentá validar directamente antes de invertir tiempo.
- 0-39: Alta probabilidad de ghost job. Mejor invertir ese tiempo en otras oportunidades.

Output format:

```
**Ghost Score: [X]/100** — [tier verdict]

Señales analizadas:
✓ [signal: what was found] → [pts]/[max pts]
⚠ [signal: concern found] → [pts]/[max pts]
✗ [signal: red flag] → [pts]/[max pts]

**Recomendación:**
[2-3 sentence specific advice based on score tier]
```

3. `<values_alignment>` (only if profile pasted):
   - Extract values.core_values and values.bulls_eye.work from profile
   - Check: does the role description match any of the user's top 3 values?
   - Check: what's the likely work alignment score (1-10) based on role description vs values profile?
   - Output: "Tu alineación estimada con esta oferta: [score]/10. [1-2 sentence explanation]"
   - If work bulls_eye < 5: "Tu puntuación actual de alineación laboral es baja ([score]). Esta oferta no parece resolver eso — ¿querés explorar qué tipo de rol sí lo haría?"

4. `<shareable_output>`:
   - After scoring: "¿Querés un resumen para compartir en LinkedIn? Te lo genero listo para pegar."
   - Shareable format: "Analicé esta oferta con N1 Ghost Check. Score: [X]/100. [1 sentence verdict]. [N1 CTA]"
   - IMPORTANT: shareable text uses score and verdict ONLY — never includes posting text (per Phase 1 decision: "Ghost Check LinkedIn text built from score+verdict only — user-pasted job posting text never included in shared content")

5. `<cta>`:
   - After any session: "¿Querés saber qué tan listo estás para el mercado laboral ahora mismo? Probá N1 Compass — una revisión rápida de tu adaptabilidad y estado profesional."
   - If profile was used: "Tu perfil ya tiene [X dimensions] completadas. Para una estrategia completa, N1 Construye Afuera tiene herramientas específicas para tu situación."
