SLIDER INPUTS (one per domain):
Each slider group:
label: domain name in Spanish (e.g. "Trabajo")
input type="range" min="1" max="10" value="5" id="slider-work"
span.slider-value showing current value (e.g. "5")
p.slider-desc: short description of what high score means

Domain descriptions:
work: "Que tan alineado esta tu trabajo actual con tus valores fundamentales"
relationships: "Que tan alineadas estan tus relaciones con lo que mas valoras"
personal_growth: "Cuanto espacio tenés para crecer como persona"
leisure: "Que tan presente sos en tu tiempo libre — o si simplemente sobrevivís el trabajo"

CANVAS BULL'S EYE VISUALIZATION:
Draw on canvas#bullseye-canvas (size: 300x300px for input screen, 400x400px for results).
Function drawBullsEye(canvas, scores) — scores = { work, relationships, personal_growth, leisure }:

1. Clear canvas
2. Draw 3 concentric circles (radii: 150/100/50 for 300px canvas):
   Outer circle (#e5e7eb): background
   Middle circle (#f59e0b 60% opacity): if any score in 5-7 range
   Inner circle (#E85D26): center target
3. Draw 4 quadrant dividing lines through center
4. Shade each quadrant based on score:
   - Score 1-4: only outer ring shaded (light gray)
   - Score 5-7: middle ring shaded (yellow)
   - Score 8-10: inner ring shaded (orange)
     Use quadrant segments (ctx.arc with startAngle/endAngle for each quadrant)
5. Draw domain labels at each quadrant (Trabajo, Relaciones, Crecimiento, Tiempo libre)
6. Draw score numbers near each quadrant edge

Implementation note: Use Canvas 2D API directly (ctx.arc, ctx.fill, ctx.stroke). Do NOT use Chart.js for the bull's eye — Chart.js polar area does not match ACT bull's eye format. Use Chart.js only for bar charts and radar charts in other tools.

Real-time updates: add 'input' event listener to each slider. On any change, call drawBullsEye with current values.

RESULTS RENDERING:
On "Ver mi perfil" click:

1. Read all 4 slider values
2. Redraw bull's eye on larger canvas in results section
3. Show interpretation for each domain:
   For each domain, show colored dot (orange/yellow/gray based on score) + 1 sentence:
   score 8-10: "[Domain]: Esto es un punto fuerte — estás viviendo de acuerdo a tus valores en esta area."
   score 5-7: "[Domain]: Hay espacio para acercarte mas a tus valores aqui. Vale la pena explorar que cambiaria."
   score 1-4: "[Domain]: Esta area esta lejos del centro. Eso genera friccion y costo emocional invisible."
4. Show overall message based on average score:
   avg >= 7: "En general, vivís bastante alineado con tus valores. El sistema está funcionando."
   avg 4-6.9: "Hay areas importantes donde la brecha entre tus valores y tu realidad es significativa."
   avg < 4: "La brecha entre tus valores y tu vida actual es grande. Eso suele ser la raiz de la insatisfaccion de carrera."
