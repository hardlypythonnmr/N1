Runway calculation logic:
Inputs:
savings: total cash savings (number)
monthly_expenses: current monthly expenses (number)
income_sources: array of { label, monthly_amount } (optional, recurring income during transition)

Monthly burn = monthly_expenses - sum(income_sources)
If monthly burn <= 0: runway is "indefinido" (income covers expenses)

Scenarios:
conservative: expenses _= 1.2 (20% expense increase — unexpected costs)
likely: expenses as entered (no adjustment)
optimistic: expenses _= 0.8 (20% expense reduction — tighten belt)

For each scenario:
adjusted_burn = scenario_expenses - sum(income_sources)
runway_months = savings / adjusted_burn (if adjusted_burn > 0)
Display: round to 1 decimal, max display = 36 (show "36+" if longer)

COP/USD toggle:
Default: COP (Colombian Pesos)
Exchange rate: 1 USD = 4200 COP (approximate — show disclaimer that rate is approximate)
Toggle converts displayed amounts and re-runs calculation
Show which currency is active with a toggle button

Chart: Chart.js bar chart, 3 bars (conservative/likely/optimistic), Y-axis = months of runway
Bar colors: conservative=gray, likely=N1 orange, optimistic=green

Financial disclaimer (per WEB-10):

CALCULATION LOGIC (pure functions):
function calculateRunway(savings, monthlyExpenses, incomeSources) {
const totalIncome = incomeSources.reduce(function(s,i){return s + (parseFloat(i.amount)||0);}, 0);
const baseBurn = Math.max(0, monthlyExpenses - totalIncome);
const scenarios = {
conservative: { label: 'Conservador', burn: baseBurn _ 1.2, color: '#9ca3af' },
likely: { label: 'Probable', burn: baseBurn, color: '#E85D26' },
optimistic: { label: 'Optimista', burn: baseBurn _ 0.8, color: '#22c55e' }
};
return Object.keys(scenarios).map(function(key) {
const s = scenarios[key];
const months = s.burn <= 0 ? null : Math.min(savings / s.burn, 36);
return { key: key, label: s.label, color: s.color, burn: s.burn, months: months };
});
}
