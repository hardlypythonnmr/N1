Flow model zones (Csikszentmihalyi):
Flow zone: challenge >= 3 AND skill >= 3 AND |challenge - skill| <= 1
Anxiety zone: challenge > skill + 1 (high challenge, low skill)
Boredom zone: skill > challenge + 1 (high skill, low challenge)

ZONE DERIVATION (pure function):
function getFlowZone(challenge, skill) {
const diff = Math.abs(challenge - skill);
if (challenge >= 3 && skill >= 3 && diff <= 1) return 'flow';
if (challenge > skill + 1) return 'anxiety';
if (skill > challenge + 1) return 'boredom';
return 'apathy';
}
