MLQ-10 Items (7-point scale: 1="Absolutamente falso" to 7="Absolutamente verdadero"):
Presence subscale (5 items — sense that life IS meaningful):
P1: "Entiendo el sentido de mi vida"
P2: "Mi vida tiene un proposito claro"
P3: "Tengo una buena comprension de lo que hace que mi vida sea significativa"
P4: "He descubierto un proposito de vida satisfactorio"
P5: "Mi vida tiene un significado claro"

Search subscale (5 items — actively SEARCHING for meaning):
S1: "Estoy buscando algo que haga que mi vida se sienta significativa"
S2: "Siempre estoy buscando el proposito de mi vida"
S3: "Estoy buscando actividades que le den proposito a mi vida"
S4: "Estoy buscando mi mision de vida"
S5: "Mi vida no tiene un proposito claro, y estoy buscando uno"

Scoring: subscale = mean of 5 items (1 decimal, 1-7 range)
Presence score = mean(P1..P5)
Search score = mean(S1..S5)

Quadrant derivation (midpoint = 4.0):
Presence >= 4 AND Search < 4 → "fulfilled" (tiene sentido, no lo busca activamente)
Presence >= 4 AND Search >= 4 → "growing" (tiene sentido y sigue construyendo)
Presence < 4 AND Search >= 4 → "seeking" (no tiene sentido claro, lo busca activamente)
Presence < 4 AND Search < 4 → "disengaged" (no tiene sentido, no lo busca)

Profile fields written:
purpose.mlq.presence = mean of P1-P5
purpose.mlq.search = mean of S1-S5
purpose.mlq.quadrant = "fulfilled"|"growing"|"seeking"|"disengaged"
purpose.completed = true
purpose.mode = "quick"

Cultural calibration (per WEB-10):
High Search (>4) in LATAM is NOT pathological — it reflects adaptive response to
economic instability and career uncertainty. Note explicitly in results.

Quadrant visualization: 2x2 Canvas diagram, axes = Presence (X) and Search (Y),
user's point plotted as a dot, quadrant regions labeled and color-coded.

   function calcMean(answers, ids) {
   const vals = ids.map(function(id) { return parseFloat(answers[id]) || 0; });
   return Math.round((vals.reduce(function(a,b){return a+b;},0) / vals.length) \* 10) / 10;
   }
   const presence = calcMean(answers, ['p1','p2','p3','p4','p5']);

2. Derive quadrant:
   function getQuadrant(presence, search) {
   const highP = presence >= 4;
   const highS = search >= 4;
   if (highP && !highS) return 'fulfilled';
   if (highP && highS) return 'growing';
   if (!highP && highS) return 'seeking';
   return 'disengaged';
   }
   const quadrant = getQuadrant(presence, search);
