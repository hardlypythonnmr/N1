RESULTS RENDERING (renderResults function):

1. Calculate subscale scores (pure functions, no mutation):
   function calcSubscale(answers, ids) {
   const vals = ids.map(function(id) { return parseFloat(answers[id]) || 0; });
   return Math.round((vals.reduce(function(a,b){return a+b;},0) / vals.length) \* 10) / 10;
   }
   const concern = calcSubscale(answers, ['c1','c2','c3']);
   const control = calcSubscale(answers, ['c4','c5','c6']);
   const curiosity = calcSubscale(answers, ['c7','c8','c9']);
   const confidence = calcSubscale(answers, ['c10','c11','c12']);

2. Show results section, hide assessment container

3. Render radar chart via Chart.js:
   new Chart(document.getElementById('radar-canvas'), {
   type: 'radar',
   data: {
   labels: ['Concern', 'Control', 'Curiosity', 'Confidence'],
   datasets: [{ label: 'Tu perfil', data: [concern, control, curiosity, confidence],
   backgroundColor: 'rgba(232,93,38,0.2)', borderColor: '#E85D26',
   pointBackgroundColor: '#E85D26' }]
   },
   options: { scales: { r: { min: 1, max: 5, ticks: { stepSize: 1 } } }, plugins: { legend: { display: false } } }
   });

   Each block: dimension name, score badge (e.g. "3.7 / 5"), 2-sentence interpretation

   Interpretation texts (static, set via textContent):
   Concern high (>=4): "Tenes una orientacion solida hacia tu futuro profesional. Planeas con anticipacion y ves las decisiones de hoy como inversiones para manana."
