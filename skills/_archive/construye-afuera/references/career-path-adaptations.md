# Career Path Adaptations — Construye Afuera Reference

> Lookup table: for each of 9 tools × 4 career paths, specific focus, key questions, output format, and language signals.
> Load this file when adapting any tool to the user's career_path.
> Internal reference — not user-facing.

---

## Career Path Values

- `stay_grow` — staying in current role/company; seeking promotion, scope expansion, or environment optimization
- `move` — changing employer; same or different function/sector
- `go_independent` — freelance, solo consulting, founding a company, portfolio career
- `pause_reset` — sabbatical, recovery, reskilling, or deliberate moratorium

## Tool 1: CV Audit

| Path           | Focus                                                                                                                              | Key Questions                                                                                                                        | Output Format                                                                                                     | Language Signals                                                                                                  |
| -------------- | ---------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------ | ----------------------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------- |
| stay_grow      | Internal resume / promotion case. Emphasis: internal scope growth, leadership, executive presence, quantified team/budget impact   | What is the promotion criteria at this company? What do decision-makers in the panel care about? What scope growth can you document? | 1-2 page internal resume or promotion case document; structured bullet format for internal panels                 | "Queremos que el panel vea la escala de tu impacto. ¿Cuántas personas influenciaste? ¿Qué presupuesto manejaste?" |
| move           | External resume. Emphasis: ATS keyword optimization, transferable skills, target role fit signal                                   | What is the target role? What keywords appear in 3-5 job descriptions for that role? What transferable skills bridge the gap?        | Reverse-chronological external CV with ATS-optimized bullets; 1 page <5 yrs, 2 pages >5 yrs                       | "El ATS va a leer esto antes que un humano. Cada bullet tiene que hablar el idioma del rol objetivo."             |
| go_independent | Portfolio / case studies. Emphasis: problem-solution-outcome format, client value language, visible expertise                      | What are 3 client problems you solved? What was the measurable outcome for the client? What expertise claim can you stake?           | Case study format: client type → challenge → approach → outcome; optionally on personal site/PDF                  | "Esto no es una hoja de vida — es un portafolio. Cada punto habla del cliente, no solo de vos."                   |
| pause_reset    | Gap narrative resume. Emphasis: positive framing of break, returning-to-workforce energy, skills maintained or acquired during gap | How long was the gap? Was it intentional or forced? What did you do/learn during it? What energy/motivation are you bringing now?    | Standard reverse-chronological with gap addressed in summary section; skills section highlights current relevance | "El gap no es el problema — cómo lo contás es lo que importa. ¿Qué aprendiste? ¿Por qué ahora?"                   |

| stay_grow      | Promotion conversation / internal interview. What's your relationship with your current manager?                                        | 3-5 STAR stories for internal achievements; script for promotion conversation opener; objection-handling responses     | "La entrevista interna tiene una capa política que la externa no tiene. ¿Quién en el panel es tu aliado?"            |

## Tool 3: Negociación

| Path           | Focus                                                                                                                                             | Key Questions                                                                                                                                                            | Output Format                                                                                          | Language Signals                                                                                                      |
| -------------- | ------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | ------------------------------------------------------------------------------------------------------ | --------------------------------------------------------------------------------------------------------------------- |
| stay_grow      | Salary raise / internal negotiation. Emphasis: timing relative to budget cycles, market data framing, how to ask without ultimatum                | When is the next salary review cycle? What is the market rate for this role in this city? What leverage do you have (competing offers, high performance)?                | 3-phase negotiation strategy: preparation → conversation → counter-offer response; conversation script | "El mejor momento para pedir un aumento es justo después de un logro visible, no en pleno review anual."              |
| move           | Offer negotiation. Emphasis: total compensation breakdown (cash+equity+benefits), multiple offer leverage, walkaway point                         | What is the full comp breakdown of the offer? What is your current total comp? Do you have other offers or can you create urgency?                                       | Counter-offer message (email template); total comp comparison table; walkaway point calculation        | "Antes de responder cualquier oferta, hacemos el breakdown completo: no solo el salario base."                        |
| go_independent | Rate setting / value-based pricing. Emphasis: market positioning, value-based vs hourly, scope creep prevention, contract terms                   | What is the market rate for this type of work in your target segment? What value does the client get from your work? How do you handle scope creep?                      | Rate card framework; project pricing formula; scope boundary template for contracts                    | "El error más común al empezar independiente es cobrar por hora. Cobrás por el resultado, no por el tiempo."          |
| pause_reset    | Re-entry negotiation. Emphasis: negotiating flexible arrangements (remote, part-time start), title matching previous level, structured onboarding | What flexible arrangement do you need to re-enter successfully? How do you frame re-entry terms without underselling yourself? What title maps to your experience level? | Flexible terms request script; re-entry negotiation talking points; title equivalency reference        | "Volvés con experiencia, no como si empezaras de cero. El framing de re-ingreso determina el piso de la negociación." |

## Tool 4: STAR Builder

| Path           | Focus                                                                                                                                                                           | Key Questions                                                                                                          | Output Format                                                                                            | Language Signals                                                                                                                     |
| -------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------ |
| stay_grow      | Internal achievement stories. Emphasis: scope of influence, peer/leadership impact, process changes, cross-functional collaboration                                             | What changed because of your work? How many people were affected? What would NOT have happened if you hadn't acted?    | 3 STAR stories emphasizing internal scope and influence; ready for promotion panel or internal interview | "En la historia interna, el resultado no siempre es un número. A veces es: '6 personas trabajan diferente por esto.'"                |
| move           | Transferable skill stories. Emphasis: skills that map to the target role, quantified impact in current sector, why it matters in the new context                                | Which skills from your current role directly transfer? What have you done that shows those skills at the target level? | 3 STAR stories written in target-role language; transferable skill mapping table                         | "Vamos a reescribir estas historias en el idioma del sector al que vas. El hecho es el mismo, el framing cambia."                    |
| go_independent | Client problem-solving stories. Emphasis: client value created, consulting-quality problem-solving, outcome for the business (not just the team)                                | What was the client/stakeholder's problem? What did you specifically do? What was the ROI for the business?            | 3 case-study-format STAR stories; client language throughout; quantified business outcome                | "Cada historia termina con '... y el cliente / la empresa obtuvo X'. Si no hay un resultado del negocio, la historia no está lista." |
| pause_reset    | Pre-gap and during-gap stories. Emphasis: include strong stories from before the gap; if there are activities during the gap (volunteering, projects, freelance), include those | What are your 3 strongest stories from before the gap? Did you do anything during the gap worth documenting?           | 3-5 STAR stories mixing pre-gap and gap-period activities; gap-period stories clearly dated              | "Si hiciste algo durante la pausa — consultaría, voluntariado, proyecto propio — tiene lugar en el STAR. No lo desestimés."          |

## Triage-to-Tool Recommendation Map

Use this map when recommending tools based on triage.situation + triage.career_path.

| situation   | career_path                         | First recommendation                                                 | Then                                              |
| ----------- | ----------------------------------- | -------------------------------------------------------------------- | ------------------------------------------------- |
| crisis      | stay_grow                           | Grow Where You Are (immediate internal stability)                    | Negociación if financial                          |
| crisis      | move                                | Layoff Response Kit (even if not laid off — crisis protocol applies) | Job Hunt                                          |
| crisis      | go_independent                      | Negociación (tarifa/rate setting for immediate income)               | Network Map                                       |
| crisis      | pause_reset                         | Layoff Response Kit                                                  | Financial runway (refer to web/runway-calculator) |
| exploration | any                                 | CTA: N1 Busca Adentro first — return here after self-knowledge work  | Most relevant single tool                         |
| growth      | stay_grow                           | Grow Where You Are                                                   | Negociación                                       |
| growth      | move (scale up/sector pivot)        | CV Audit → STAR Builder → Interview Prep                             | Network Map                                       |
| growth      | move (function pivot/career change) | STAR Builder → CV Audit → Profile Builder                            | Interview Prep                                    |
| growth      | go_independent                      | Network Map → Negociación (rates)                                    | Profile Builder                                   |
| growth      | pause_reset (education/reskill)     | CV Audit (gap narrative) → Profile Builder                           | Job Hunt                                          |
| growth      | pause_reset (sabbatical/recovery)   | Grow Where You Are (optimize before leaving)                         | then Busca Adentro                                |

**Urgency override:** If urgency is `immediate`, always prioritize the tool that produces direct job-search or income output first, regardless of path theory.

## specific_move Micro-Adaptations

When `triage.specific_move` is available, apply these additional micro-adaptations:

| specific_move                | Additional adaptation                                                      |
| ---------------------------- | -------------------------------------------------------------------------- |
| Promotion (vertical)         | Grow Where You Are Module B (promotion case) is primary                    |
| Lateral transfer             | Grow Where You Are Module A (opportunity scan) is primary                  |
| Role expansion               | Grow Where You Are Module B + Negociación combo                            |
| Environment optimization     | Grow Where You Are Module C + Negociación                                  |
| Scale up (bigger company)    | CV Audit (ATS + scale) + Interview Prep (larger company culture)           |
| Scale down (smaller company) | CV Audit (ownership narrative) + STAR Builder (breadth stories)            |
| Sector pivot                 | STAR Builder (transferable) + CV Audit + Interview Prep                    |
| Function pivot               | Busca Adentro first (CTA) — high self-knowledge requirement                |
| Complete career change       | Busca Adentro first (CTA) — highest self-knowledge requirement             |
| Return to workforce          | CV Audit (gap narrative) + Interview Prep (re-entry) + Layoff Kit protocol |
| Freelance / solo consulting  | Negociación (rates) + Network Map (client acquisition) + Profile Builder   |
| Found a company              | Busca Adentro first — identity work needed for founder path                |
| Join early-stage startup     | STAR Builder + Interview Prep (startup cultural fit signals)               |
| Portfolio career             | Network Map + Profile Builder (multi-expertise positioning)                |
| Strategic sabbatical         | Grow Where You Are (optimize before leaving)                               |
| Recovery period              | Layoff Response Kit structure applies even if not laid off                 |
| Education / reskilling       | CV Audit (skills gap) + Job Hunt (future-state role targeting)             |
