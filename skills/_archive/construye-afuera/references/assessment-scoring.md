# Assessment Scoring Reference — Construye Afuera

> Rubrics and signal libraries for CV Audit, Ghost Check (job postings), and Interview quality assessment.
> Load this file when running CV Audit, Job Hunt (ghost check), or Interview Prep tools.
> Internal reference — not user-facing.

---

## Part 1: CV Audit Rubric — 10 Dimensions

### Scoring overview

Each dimension is scored 0-10. Total possible: 100 points.

- **80-100:** Ready to send — minor polish only
- **60-79:** One revision needed — address highest-gap dimensions
- **40-59:** Significant work — systematic rewrite required
- **Below 40:** Major reconstruction — likely needs full rewrite

### Dimension scoring guide

---

**Dimension 1: Impact Metrics (0-10)**

Does each bullet contain a quantified result?

| Score | Description                                                                |
| ----- | -------------------------------------------------------------------------- |
| 9-10  | Every bullet has a specific number, %, $, or clear before/after comparison |
| 7-8   | Most bullets have metrics; 1-2 are descriptive only                        |
| 5-6   | Some metrics present; many bullets describe duties, not outcomes           |
| 3-4   | Few metrics; mostly duty descriptions ("responsible for X")                |
| 0-2   | No metrics; pure job description language throughout                       |

**Improvement action:** "Reescribí cada bullet con este formato: [Verbo de acción] + [Qué hiciste] + [Resultado medible]. Ejemplo: 'Reducí el tiempo de deployment de 4 horas a 45 minutos automatizando el pipeline de CI/CD.'"

---

**Dimension 2: ATS Keyword Density (0-10)**

Are role-specific keywords present in the first half of the CV, in the experience bullets, and in the skills section?

| Score | Description                                                                          |
| ----- | ------------------------------------------------------------------------------------ |
| 9-10  | Keywords match 80%+ of the target job description; naturally integrated, not stuffed |
| 7-8   | Key technical skills and role-specific terms present; minor gaps                     |
| 5-6   | Some keywords present; missing key terms that appear in multiple job postings        |
| 3-4   | Few role-specific keywords; generic language dominates                               |
| 0-2   | Keywords absent; CV could apply to any role                                          |

**How to identify keywords:** Pull 3-5 job descriptions for the target role. Highlight terms that appear in 3+ of them. These are must-have keywords.

**Improvement action:** "Comparó tu CV con 3 ofertas reales del rol objetivo. Los términos que aparecen en todas las ofertas y no están en tu CV son las brechas de keywords más críticas."

---

**Dimension 3: Narrative Coherence (0-10)**

Does the CV tell a logical career progression story? Can a reader understand where you started, where you've been, and why this role makes sense as the next step?

| Score | Description                                                                           |
| ----- | ------------------------------------------------------------------------------------- |
| 9-10  | Clear progression; each role builds on the last; target role is the obvious next step |
| 7-8   | Generally coherent; one or two transitions could be better explained                  |
| 5-6   | Some logic evident but gaps or pivots that raise questions without explanation        |
| 3-4   | Disconnected history; progression unclear                                             |
| 0-2   | Random-seeming history; reader has no idea what the person is building toward         |

**Improvement action:** "El CV necesita un hilo narrativo. La sección de resumen/objetivo puede ser el lugar para tejerlo explícitamente: '[Background] con X años construyendo [capacidad específica] — ahora buscando [tipo de rol].'"

---

**Dimension 4: Length and Format (0-10)**

Is the length appropriate for experience level? Is the format clean, consistent, and ATS-passable?

| Score | Description                                                                                    |
| ----- | ---------------------------------------------------------------------------------------------- |
| 9-10  | Correct length (1 page ≤5 yrs; 2 pages >5 yrs); single-column; consistent spacing; clean fonts |
| 7-8   | Mostly correct; minor formatting inconsistencies                                               |
| 5-6   | Length slightly off OR formatting issues that may affect ATS parsing                           |
| 3-4   | Significantly wrong length (under-filled 1 page with 8 yrs experience, or 4-page over-bloated) |
| 0-2   | Two-column layout, text boxes, tables, graphics — likely to fail ATS                           |

**LATAM-specific note:** Two-column Canva/Word templates are extremely common in Colombia — these typically fail ATS. Flag this explicitly.

**Improvement action:** "El formato de dos columnas es muy común en plantillas colombianas pero rompe la mayoría de los sistemas de selección automática. Recomiendo convertir a una sola columna en formato simple."

---

**Dimension 5: Headline / Professional Summary (0-10)**

Is the headline or summary specific, compelling, and targeted to the role?

| Score | Description                                                                        |
| ----- | ---------------------------------------------------------------------------------- |
| 9-10  | Specific expertise claim + target role type + differentiator; reads in <10 seconds |
| 7-8   | Mostly specific; could be sharper; misses one element                              |
| 5-6   | Present but generic ("Profesional con X años de experiencia...")                   |
| 3-4   | Very generic or absent                                                             |
| 0-2   | Absent; no summary section at all                                                  |

**Generic phrases to avoid:** "Profesional con amplia experiencia", "Busco una empresa donde pueda desarrollarme", "Trabajo en equipo", "Orientado a resultados".

**Improvement action:** "El headline debería responder en 2 líneas: ¿Qué hacés exactamente? ¿Para qué tipo de rol o empresa? Ejemplo: 'Senior Backend Engineer especializado en arquitectura de microservicios con Java/Kotlin — 6 años en productos fintech y outsourcing para clientes US.'"

---

**Dimension 6: Skills Match to Target Role (0-10)**

Does the CV explicitly list skills that match the target role requirements?

| Score | Description                                                                                |
| ----- | ------------------------------------------------------------------------------------------ |
| 9-10  | Skills section directly maps to target role requirements; technical and soft skills listed |
| 7-8   | Good coverage; minor gaps                                                                  |
| 5-6   | Skills section present but not targeted; or targeted but missing key skills                |
| 3-4   | Minimal skills section; or skills don't match target                                       |
| 0-2   | No skills section; or completely mismatched                                                |

**Improvement action:** "Comparó tu sección de habilidades con los requisitos de 3 ofertas reales. Añadí las herramientas y tecnologías que aparecen consistentemente y que vos dominás pero no tenés listadas."

---

**Dimension 7: Value Alignment (0-10) [Optional but high-signal]**

Does the CV signal what the person values in their work? Is there evidence of a coherent career direction that reflects intentionality?

_This dimension is only scored when the user has a profile with values data. Optional but powerful — especially for roles where culture fit is explicit in the hiring process._

| Score | Description                                                                              |
| ----- | ---------------------------------------------------------------------------------------- |
| 9-10  | CV communicates a clear professional point of view; role choices reflect coherent values |
| 7-8   | Some alignment visible; could be more intentionally surfaced                             |
| 5-6   | Alignment present but implicit; reader has to infer it                                   |
| 3-4   | No clear values signal; could be anyone                                                  |
| 0-2   | CV signals misalignment (person communicating values opposite to their stated ones)      |

---

**Dimension 8: Strength Evidence (0-10) [Optional but high-signal]**

Are the user's signature strengths visible in the achievement descriptions?

_Only scored when user has provided VIA strengths or has articulated their top strengths._

| Score | Description                                                                      |
| ----- | -------------------------------------------------------------------------------- |
| 9-10  | Signature strengths clearly visible across multiple bullets and experience items |
| 7-8   | Strengths evident in some sections                                               |
| 5-6   | Partially visible; strengths present but buried                                  |
| 3-4   | Strengths not evident; generic achievement descriptions                          |
| 0-2   | CV actively contradicts stated strengths                                         |

---

**Dimension 9: Cultural Calibration (0-10)**

Is the CV appropriate for the target market? Does it match the cultural expectations of the employer?

| Score | Description                                                                  |
| ----- | ---------------------------------------------------------------------------- |
| 9-10  | Fully calibrated: language, format, and tone match target market conventions |
| 7-8   | Mostly calibrated; minor mismatches                                          |
| 5-6   | Calibrated for one market but applying to another without adaptation         |
| 3-4   | Significant mismatch (e.g., LATAM-format CV applying to US market)           |
| 0-2   | Completely wrong format for target market                                    |

**LATAM → International calibration notes:**

- Photo: standard in Colombian CVs; omit for US/EU applications
- Marital status, age, ID number: common in Colombian CVs; omit for US/EU (legal and cultural issue)
- Spanish-language CV: only if applying to Spanish-speaking companies; translate for international
- References section: "Referencias disponibles a solicitud" is Colombian standard; omit for US where implied

---

**Dimension 10: Contact Information / CTA (0-10)**

Is contact information complete, clean, and professional?

| Score | Description                                                                                        |
| ----- | -------------------------------------------------------------------------------------------------- |
| 9-10  | LinkedIn URL (personalized), professional email, location (city, not full address), phone optional |
| 7-8   | Most info present; minor issues (generic LinkedIn URL, old email)                                  |
| 5-6   | Missing one key element (no LinkedIn, no location)                                                 |
| 3-4   | Multiple elements missing                                                                          |
| 0-2   | No contact info or unprofessional (hotmail from 2003, personal blog unrelated to career)           |

**LinkedIn URL:** Most users have the default `linkedin.com/in/johndoe-123456`. The custom URL (`linkedin.com/in/juanperez`) looks cleaner and is easy to set.

---

## Part 2: Ghost Check Signal Library

### Purpose

Used by the Job Hunt tool to evaluate job posting quality before the user invests time in an application. Also surfaced when a user pastes a job description for analysis.

Signals are categorized as Red, Yellow, or Green.

---

### Red Flags — high likelihood the posting is problematic

**Posting quality red flags:**

- Posting age >90 days without being filled (check LinkedIn "posted X days ago" or refresh date)
- Reposted multiple times (original posting date vs "re-listed" date is a signal)
- Generic role title that could mean anything ("Business Development", "Project Manager" without context)
- Zero company information — no About section, no company description, no URL
- Mass posting: same company posting 10+ identical roles on the same day
- Copied-and-pasted boilerplate that doesn't describe any specific team or project

**Compensation red flags:**

- "Salary: competitive" with no range — employer hiding compensation for a reason
- No compensation information at all in markets where ranges are now standard (US, some EU)
- The posted range is extremely wide — e.g., "$40K–$120K" — signals no real definition of the role

**Process red flags:**

- Requests unusual personal information upfront: ID number, bank info, photo, personal documents
- Asks for unpaid work, "take-home projects" that would take >4 hours, or spec work
- No mention of a hiring timeline or process steps
- "No experience needed" for a role that clearly requires experience
- Vague hiring manager or "hiring committee" with no named contacts

**Company red flags:**

- Company has no LinkedIn presence or very new company page (< 1 year) with no employees listed
- Reviews on Glassdoor or comparable tools are primarily 1-2 stars with similar complaints
- Multiple rounds of layoffs in the past 12 months (cross-check LinkedIn news)
- Job has been open for 6+ months — typically signals: budget frozen, impossible requirements, or revolving door role

---

### Yellow Flags — investigate before applying

- Salary range is very wide (3x span or more): `$50K–$150K` suggests role is not well-defined
- Location listed as "flexible/hybrid" without specifying what that means in practice
- "We're growing fast" as the main employer value proposition without specifics
- "You'll wear many hats" — sometimes a real opportunity; sometimes "we'll overload you"
- Role reports to "VP" or "C-suite" directly in a large company — may signal instability or that the role is low-leverage
- No mention of the team structure or how many people do this work
- Repeated emphasis on "culture fit" without describing the culture specifically
- "We work hard and play hard" — cultural signal worth investigating further before accepting an offer

---

### Green Flags — signals quality posting

**Compensation:**

- Specific salary range (narrow, e.g., "$70K–$90K" or "COP 8M–12M/month")
- Total comp breakdown mentioned (base + bonus + equity + benefits)
- Transparent about remote work stipend or equipment policy

**Role clarity:**

- Named hiring manager or specific team mentioned
- Clear description of day-to-day responsibilities
- Explicit "what success looks like at 3, 6, 12 months"
- Specific tech stack, tools, and methodologies listed

**Process:**

- Clear multi-step process described (e.g., "screening call → technical interview → panel interview → offer")
- Timeline explicitly stated ("we aim to close this role in 4 weeks")
- Recruiter or HR contact named

**Company:**

- Recent posting (< 30 days)
- Active company LinkedIn page with named employees and recent activity
- Multiple positive signals: company news, product launches, recent funding (for startups)
- Consistent positive employer branding across Glassdoor, LinkedIn, and the posting itself

---

When running Ghost Check as a conversational tool, rate the posting:

## Part 3: Interview Quality Signals

### Purpose

Used by the Interview Prep tool to help users evaluate the quality of the interview process they're experiencing — not just prepare to perform well, but to evaluate whether the company/role is worth the investment.

---

### Strong signals from the interviewer / process

**Signals the role is well-defined:**

- Interviewer describes specific challenges the person in this role will face in the first 90 days
- Team structure is explained: who you'd work with, report to, and what other roles exist
- Clear timeline for the hiring decision is provided upfront
- Consistent story across multiple interviewers (if panel)

**Signals the company respects the candidate:**

- Process is explained at the beginning: "This interview has two parts..."
- Interviewer is on time and prepared (has read your CV)
- Behavioral questions are specific ("tell me about a time when...") rather than generic ("tell me about yourself")
- Candidate has time to ask questions at the end
- Recruiter follows up promptly after each stage

**Signals of good culture for execution:**

- Interviewers can describe a specific recent success the team had
- Disagreements are described with "we discussed" rather than "my manager decided"
- Interviewers speak positively about colleagues without over-praising
- Specific examples of how the company supports professional development

---

### Weak signals — worth asking follow-up questions

- "We're still defining the role" — ask what decisions are pending and the timeline
- Interviewer can't describe what a typical day looks like in this role
- No clear timeline for the process or decision — ask directly
- Generic questions only ("What are your strengths and weaknesses?") — suggests unprepared interviewer or poorly structured process
- Vague team structure descriptions — ask specifically: "Who would I collaborate with daily?"
- "We'll let you know soon" without a specific date

---

### Red flags in the interview process

**Legal / ethical:**

- Questions about salary history (illegal in some jurisdictions, including many US states)
- Questions about family plans, marital status, nationality (illegal in many contexts)
- Pressure to decide quickly without time to review an offer

**Process red flags:**

- Interviewer is dismissive of questions or makes the candidate feel like asking questions is inappropriate
- No mention of compensation range at any point in a final-stage process
- "You'll start at the bottom but we promote fast" — vague promise, investigate
- Inconsistent information from different interviewers about the role or team
- Unpaid homework assignment disproportionate to the interview stage

**Culture red flags:**

- Interviewer speaks negatively about the previous person in this role
- "We work hard here" as a virtue statement without acknowledging balance
- Decision-making described as "the CEO decides everything" without context about company size
- Multiple interviewers can't explain why someone would want this specific role

---

### Interview quality self-assessment template

After each interview stage, the user can use:

```
Evaluación de la entrevista — [Empresa / Rol / Fecha]

¿El rol está bien definido? [Sí / Parcialmente / No]
¿El proceso fue organizado y respetuoso? [Sí / Parcialmente / No]
¿El entrevistador estaba preparado? [Sí / Parcialmente / No]
¿Pude hacer preguntas? [Sí / No]
¿La compensación fue mencionada o es accesible? [Sí / No]

Señales positivas: [list]
Señales de alerta: [list]

¿Vale la pena continuar? [Sí / Necesito más información / No]
Próximos pasos: [acción]
```
