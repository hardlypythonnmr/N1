Qué cambió en esta versión

La pregunta vieja era "¿Por dónde empezás?" con 3 preguntas de triage. Producía autoconocimiento, no entrevistas.

La nueva versión arranca donde estás bloqueado. Una sola pregunta. Cinco caminos. Cada uno es una skill AI-native específica, no orientación genérica. La investigación validada (CAAS, MLQ, ACT, VIA) vive dentro de Prepararse — no es el gate de entrada, es una palanca cuando la necesitás.

Spanish-primary, LATAM-rooted. Dos skills gratis para siempre (Conseguir y Ganar Entrevistas) en GitHub. Tres skills en la biblioteca de práctica — visibles acá, acceso por definirse.

Cómo empezar

Opción A · Claude skills gratis. Clonás el repo, copiás skills/get-interviews/ o skills/win-interviews/ en un Claude Project. Frase trigger del SKILL.md, listo.

Opción B · herramientas web. El panel derecho tiene 3 mini-versiones web de las skills. Sin cuenta, sin instalación, sin costo. Guardan tu perfil en localStorage.

Opción C · ChatGPT GPTs. Próximamente en la tienda de GPTs de OpenAI.
