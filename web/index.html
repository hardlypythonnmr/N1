Qué cambió en esta versión

La pregunta vieja era "¿Por dónde empezás?" con 3 preguntas de triage. Producía autoconocimiento, no entrevistas.

La nueva versión arranca donde la persona está bloqueada. Una sola pregunta. Cinco caminos. Cada uno es una skill AI-native específica, no orientación genérica. La investigación validada (CAAS, MLQ, ACT, VIA) vive dentro de Prepararse — no es el gate de entrada, es una palanca cuando la necesitás.

Spanish-primary, LATAM-focused. Gratis. Open source. Código en GitHub. Instalás las skills en tu Claude Project, o usás las herramientas web acá debajo sin cuenta.

Cómo usar las skills

Opción A · Claude. Clonás este repo, copiás la carpeta skills/[nombre]/ dentro de un Claude Project. El skill se activa al usar las frases trigger. Todas las skills son gratuitas.

Opción B · ChatGPT. Hay GPTs con versiones compactas en el GPT Store (busca "N1"). Funcionalidad reducida por budget de tokens, pero entrada libre.

Opción C · Web. Las herramientas del panel derecho funcionan acá mismo, sin instalación. Te piden datos con preguntas validadas y te devuelven un perfil YAML que podés pegar en Claude o ChatGPT para continuar.
